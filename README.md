# Phishing-Awareness-2

This project demonstrates the risks of QR-based phishing attacks (Quishing) through an interactive web simulation.

## 🚀 Features
- Interactive QR phishing simulation
- User decision-based outcome (safe vs attacked)
- Clean and responsive UI
- Beginner-friendly cybersecurity awareness project

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📱 Live Demo
https://github.com/lucifer55520/Phishing-Awareness-2

## ⚠️ What is Quishing?
Quishing (QR Phishing) is a cyber attack where attackers use malicious QR codes to redirect users to fake websites to steal sensitive data.

## 🎯 Purpose of Project
This project is built to:
- Spread awareness about phishing attacks
- Demonstrate real-world cyber threats
- Improve practical frontend and security concepts

## 📦 Installation (For Local Use)
1. Clone the repository
2. Open `index.html` in browser
